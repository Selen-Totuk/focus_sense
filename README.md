# OdakTakip – Focus Tracking App

---

## English

**OdakTakip** is a Python application that tracks screen time, posture, blink count, and focus percentage while you are in front of your screen. It uses OpenCV and Tkinter for real-time tracking and visualization.

### Features

- Live camera feed with focus and posture tracking  
- On-screen and off-screen time measurement  
- Blink count  
- Posture analysis (Good / Bad)  
- Focus percentage  
- TR/EN language support  
- Dark and light themes  
- Simple gaze history map visualization  

### Requirements

- Python 3.x  
- OpenCV (`opencv-python`)  
- PIL / Pillow (`pillow`)  
- Tkinter (comes with Python)

### Installation

Usage
